# 电影频道

## 使用以下两个npm命令即刻运行查看项目概况
```
npm install
npm run serve
```

## 功能清单
**首页** <br>
- [ ] 搜索电影 <br>
- [x] 豆瓣top50展示 <br>

**热映榜单** <br>
- [ ] 用户定位城市 <br>
- [x] 北美热映票房榜单 <br>
- [ ] 用户定位城市热映票房榜单 <br>

**我的** <br>
- [ ] 个人信息 <br>
- [ ] 收藏电影 <br>
- [ ] 看过的电影 <br>
