# 电影频道

## 使用以下两个npm命令即刻运行查看项目概况
```
npm install
npm run serve
```

## 功能清单
**首页** 
-[ ] 搜索电影
-[X] 豆瓣top50展示

**热映榜单** 
-[ ] 用户定位城市
-[X] 北美热映票房榜单
-[ ] 用户定位城市热映票房榜单

**我的**
-[ ] 个人信息
-[ ] 收藏电影
-[ ] 看过的电影
